Array com todos os Elementos

querSelectorAll - Traz um array de todos os elementos
var teste = document.querySelectorAll(".elemento");

for (var i = 0; teste.length; i++){

Bucar Elementos

var valorx = document.querySelector(.titulo); - . (ponto) realiza busca pela classe do elemento
var valorx = document.querySelector(#titulo); - # (hasgtag) realiza busca pelo ID do elemento

Pegando valores de uma tabela e somando var y = document.querySelector("#conteudo do elemento TR");
var z = y.querySelector(".conteudo elemento TD");
var val = z.textContent + y.textContent; - somando os valores

Adicionar atributos aos Elementos

var valorx = document.querySelector(.titulo); - busca pela classe
valorx.classList.add("nome_de_uma_classe_qualquer_ou__de_um_css");
