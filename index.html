Template Literals

Desta forma o navegador não sabe o que é Desconsiderar os pontos dentro das tags, fiz isso apenas pra conseguir exibi-las no navegador 

              

              const conteudoLinha =
                  <.tr>
                  <.td>681561651<./td>
                  <.td>Fulano<./td>
                  <./tr>
Usando crase o navegador consegue interpretar 
              

              const conteudoLinha =
                  ` <.tr>
                      <.td>681561651<./td>
                      <.td>Fulano<./td>
                  <./tr> `

Template Literals - acessando dados de um Objeto Array

Abaixo temos o array JS 
              

              let dados = [
              {
                  cpf: 9816516165,
                  nome: "fulano"
              }                
                          ];
Para usar os dados dele da forma abaixo não funciona, pois ele interpreta tudo como string 
              

              const conteudoLinha =
              ` <.tr>
                  <.td>dados[0].cpf<./td>
                  <.td>dados[0].nome<./td>
              <./tr> `
Para informar que é javascript usar como abaixo 
              

              const conteudoLinha =
              ` <.tr>
                  <.td>${dados[0].cpf}<./td>
                  <.td>${dados[0].nome}<./td>
              <./tr> `
